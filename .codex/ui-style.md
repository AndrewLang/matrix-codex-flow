VibeFlow UI Style Contract

Overall feel:
- Dark, premium, neumorphic + glass
- Soft shadows, no hard borders
- Calm, airy spacing
- Rounded large corners (rounded-2xl / rounded-3xl)
- Depth through shadows, not outlines

Panels:
- bg-slate-900/60 with backdrop-blur
- shadow-[8px_8px_20px_rgba(0,0,0,0.6),_-8px_-8px_20px_rgba(255,255,255,0.03)]
- No borders

Chat / Cards:
- Look like floating message bubbles
- Generous padding (px-4 py-3 or larger)
- Max width around 60â€“70%
- Space between items: space-y-4 or more

Composer / Inputs:
- Inset (shadow-inner)
- Rounded-2xl
- Feels like chat input, not form field

Timeline / Steps:
- Vertical execution line
- Step cards attached to the line
- Colored badges on the left
- Feels like a pipeline

Buttons:
- Rounded-full or rounded-xl
- Subtle hover glow
- No harsh outlines

Spacing:
- Airy, calm, premium
- Avoid dense layouts

Typography:
- Clean, modern, slightly larger than default
- Subtle text colors (text-slate-300, text-slate-400)
